食品廃棄ゼロへ
CONCEPT / local

スーパーで捨てられる野菜を、
豚の飼料として自動でマッチング。

毎日トン単位で廃棄されるスーパーの野菜くずや惣菜。一方で、飼料価格の高騰に苦しむ畜産農家。この分断された情報を繋ぎ、焼却費用と餌代の双方を大幅に削減するエコ循環。

UI SYSTEM ONLINE

[ PROTOTYPE CONCEPT RENDER ]

廃棄量IoT自動計量システム

スーパーのバックヤードのゴミ箱にAIスケールを設置。「本日キャベツ外葉50kg発生」といった情報をデータベースに自動登録。

畜産農家への即時ルート配送

発生した廃棄物（エコフィード）を欲しがっている近隣の養豚場とマッチング。提携トラックが回収ルートを自動最適化。

サステナブル証明書発行

スーパー側は「廃棄ゼロ」のCSRレポートをボタン一つで出力。ESG投資へのアピール材料として活用可能。
